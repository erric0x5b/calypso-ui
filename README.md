# Calypso UI

BlueOS extension for the DeepEx Calypso ROV control interface.

The extension provides the vehicle dashboard, power SCADA, telemetry widgets,
mission logging, video stream display, sonar controls, diagnostics, and setup
tools used by the Calypso UI backend.

## BlueOS Installation

Use the extension collection manifest published from this repository, or install
the image manually:

```text
ghcr.io/erric0x5b/calypso-ui:0.4.10
```

The UI is served by the extension at `/ui`.
